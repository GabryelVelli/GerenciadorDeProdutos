# Sistema de Gestão de Produtos

Um sistema simples de Gestão de Produtos para Adicionar, Visualizar e Excluir utilizando: 
- **Backend**: Node.js com Express
- **Banco de Dados**: SQL Server
- **Frontend**: Knockout.js e JQuery
- **Estilo**: CSS simples

## Funcionalidades:
- Adicionar produtos
- Deletar produtos
- Listar produtos
